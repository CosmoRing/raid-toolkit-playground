Resume Helper

  • Total: Total of food champions in the current rank.
  • Max Level: Total of food champions in the current rank at max level.
  • Chickens: Number of chickens of rank. Can be ignored using the chicken icon (right side) for a specific rank.
    If you have more chickens than food champions, the number of champions will be the limiting factor.
  • Can Upgrade: Number of total food champions that can be upgraded to the next rank. If it's set to use lower rank champions, it uses the number on Can Upgrade from the previous rank and sums it with the current. You can ignore previous rank using the button next to the chicken.
  • Missing Max Level: You still need X food champions at max level on the rank to achieve the number in Can Upgrade.
  • Missing for Next Upgrade: You still need X food champions on the current rank to upgrade one more. If it's set to use lower rank champions, you will see the number of food champions "needed" sorted from current rank to lower rank.
    Ex: 3/9/26
    For the next upgrade to be available you need: 3 .
    To get these you need: 9 or 26
